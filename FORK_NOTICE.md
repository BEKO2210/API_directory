# Why This Fork Exists

## Background

The original [public-apis/public-apis](https://github.com/public-apis/public-apis) repository was one of the most valuable open-source projects for developers — a community-curated list of **free public APIs**. With over 300,000 stars, it became a go-to resource for anyone building software.

In 2021, the project was taken over by **APILayer**, a commercial company. The original maintainers — including [Matheus Felipe (matheusfelipeog)](https://github.com/matheusfelipeog) and others who built and nurtured the project — were unable to continue maintaining it due to this takeover.

You can read the full story from one of the original maintainers here:
**[public-apis/public-apis#3104](https://github.com/public-apis/public-apis/issues/3104)**

Since then, the repository has been largely unmaintained. Pull requests go unmerged, issues go unanswered, and the community that built this resource has no way to contribute anymore.

## What This Project Does

**API Hub** started as a visual frontend to make the public-apis data easier to browse. After speaking directly with Matheus Felipe (original maintainer), who confirmed this use is welcome and encouraged, we decided to take the next step:

**This repository now serves as a community-maintained continuation of the public-apis project.**

What this means:
- We maintain a **local copy of the full API dataset** alongside the original source
- **Anyone can contribute** — report broken APIs, suggest new ones, or fix data directly
- Issues reported on the website are **automatically processed** on each workflow run
- The data stays **open and accessible** to everyone

## Giving Back

**This repository and all rights are offered to the original contributors.**

If any of the original maintainers (Matheus Felipe, Dave Machado, or others) want to:
- Take over this repository
- Co-maintain it
- Transfer it to a new organization

They are welcome to do so at any time. Just open an issue or contact [@BEKO2210](https://github.com/BEKO2210).

The goal was never to "own" this data — it's to keep it alive and accessible for the developer community.
